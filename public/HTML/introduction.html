Introduction

Seeing this text means that you know about our secret society. I have unleashed something terrible onto this world and I'm not sure how to fix it. Details are not important. This website is a puzzle site designed to check the problem solving and fortitude of potential members. If you complete all 6 puzzles, you will join my secret society to help stop what I've unleashed.

When you have the answer to the puzzle, you will type the answer in the URL.
EX: localhost:3000/puzzle-answer (replace "puzzle-answer" with the actual answer).

If you understand how dire the situation is (or if you're simply bored and want to pass time), click the button to start the puzzles.
